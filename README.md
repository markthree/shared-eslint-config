# shared-eslint-config

通用的 eslint 配置

<br />

## 动机

eslint 的配置参考

<br />

## 特性

- `commit` 前自动 `lint`
- `vscode` 插件自动推荐
- `vscode` 保存自动 `lint`
- `prettier` 代理代码风格
- 支持 `editorConfig` 的通用配置

<br />

## 支持列表

- jsx
- node
- es2021
- browser
- vue3-sfc
- typescript
